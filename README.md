|Refactor|기존의 코딩이 수정되었을 때|성능 향상 X|
|Fix|오류를 수정했을 때||
|Close|개발 이슈를 완료했을 때||
|Docs|MARKDOWN 문서 수정||
|Remove|파일 삭제했을 때||
